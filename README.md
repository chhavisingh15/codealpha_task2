# 🌍 Language Translation Tool

A Python-based Language Translation Tool that translates text between multiple languages using a translation API. The application provides a simple and user-friendly interface, allowing users to quickly convert text from one language to another with accurate results.

## Features
- Translate text between multiple languages
- Fast and accurate translations
- Simple and intuitive user interface
- Supports various languages
- Lightweight and easy to use

## Technologies Used
- Python
- Translation API (Googletrans/Google Translate API)
- Requests Library

## Installation
1. Clone this repository.
2. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
   python app.py
   ```

## Usage
- Enter the text you want to translate.
- Select the source and target languages.
- Click the translate button to view the translated text instantly.
